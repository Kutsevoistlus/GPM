# Setup
Pärast `npm installi` alusta backendi kasutades `npm run start`
või `npm back/server`  
  
`.env` file tuleb luua `.env.sample` faili põhjal.  
`PORT` on vajalik täita lehe töötamiseks ning `SENGDGRID_API_KEY` on vajalik emailide saatmiseks.
